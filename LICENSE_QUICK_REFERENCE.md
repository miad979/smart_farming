## 📖 License Files Explained

### LICENSE
**What:** Standard MIT License  
**Size:** 21 lines  
**Purpose:** Main license terms  
**Required:** Always  
**Modify:** Never  

### NOTICE
**What:** Legal notices and disclaimers  
**Size:** ~100 lines  
**Purpose:** Apache 2.0 compliance, disclaimers  
**Required:** When redistributing  
**Modify:** Never  

### ATTRIBUTIONS.md
**What:** Third-party component credits  
**Size:** ~20 lines  
**Purpose:** Credit shadcn/ui, Unsplash  
**Required:** If using UI components  
**Modify:** Never (platform-protected)  

### THIRD_PARTY_LICENSES.md
**What:** Full license texts of dependencies  
**Size:** ~500 lines  
**Purpose:** Complete legal transparency  
**Required:** When redistributing  
**Modify:** Only to add new dependencies
