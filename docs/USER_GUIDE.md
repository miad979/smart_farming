## Getting Started

### First Time Opening the App

1. **Open the Application**
   - Visit the Smart Farming web app URL
   - No installation required - works in any browser
   - Optimized for mobile phones (Android)

2. **Choose Your Language**
   - Default: Bengali (বাংলা)
   - Toggle to English anytime from Profile
   - Language preference saved automatically

3. **View the Dashboard**
   - You start as a **Guest** (no login required)
   - Access all main features immediately
   - See weather, alerts, and quick stats

4. **Complete Onboarding (Optional)**
   - Dismiss the welcome banner after reading
   - Won't show again once dismissed

### Understanding Guest Mode vs Logged In

**Guest Mode (Default):**
- ✅ Check disease detection
- ✅ View irrigation status
- ✅ Check market prices
- ✅ View weather
- ❌ Cannot save results
- ❌ Cannot request consultations
- ❌ No cloud sync

**Logged In Mode:**
- ✅ All guest features
- ✅ Save disease detection results
- ✅ Request expert consultations
- ✅ Maintain crop logs
- ✅ Cloud sync across devices
- ✅ Consultation history

### How to Login

1. Go to **Profile** (bottom right icon)
2. Click **"Login with OTP"** button
3. Enter your phone number
4. Enter the OTP code (sent via SMS)
5. You're now logged in!

**Note:** In demo mode, any 6-digit code works as OTP.

### Disease Detection

#### How to Detect Crop Disease

1. **Navigate to Disease Detection**
   - Tap Camera icon (bottom nav)

2. **Take or Upload Photo**
   - **Take Photo:** Use your phone camera

4. **Read Advisory**

5. **Listen to Advisory**
   - Tap the Speaker icon

6. **Save or Share Results**

   - 🔗 **Share:** Send to family/experts

   - Use Chat Assistant for quick questions
   - Quick reply buttons for common queries

8. **Request Expert Consultation** (Login Required)
   - Click "Need Expert Opinion?"

### Irrigation Management

#### Monitor Soil Moisture

**Moisture Level Display:**
- **Green (70%+):** Healthy - No action needed
- **Blue (50-69%):** Moderate - Monitor
- **Orange (<50%):** Low - Water soon

#### Auto-Watering Mode

**How to Enable:**
1. Go to Irrigation page
2. Toggle "Auto Mode" switch to ON (green)
3. System waters automatically when moisture drops

**How It Works:**
- Monitors soil moisture continuously
- Waters when below threshold (set in policy)
- Shows next watering time
- Saves water usage history

#### Manual Watering

**When to Use:**
- Auto mode is OFF
- Need immediate watering
- Testing the system

**How to Water:**
1. Click **"Water Now"** button (big blue button)
2. Watering starts immediately
3. Duration based on policy settings
4. View amount on screen

#### Irrigation Policy Settings

**What It Controls:**
- Which crop you're growing
- Moisture threshold (when to water)
- Water amount per session
- Watering duration

**How to Change:**
1. Click "Irrigation Policy" to expand
2. View current settings
3. Note: Editing requires admin/expert assistance

## Troubleshooting

### Disease Detection Not Working

**Problem:** Can't take photo or upload image

**Solutions:**
1. Check camera permission (Settings → App Permissions)
2. Ensure good lighting
3. Try upload instead of camera
4. Check internet connection
5. Refresh the page

**Problem:** AI analysis taking too long

**Solutions:**
1. Wait 30 seconds
2. Check internet speed
3. Try smaller image size
4. Retry scan

---

### Irrigation Shows Wrong Data

**Problem:** Moisture level seems incorrect

**Solutions:**
1. Check if sensor is working
2. Refresh the page
3. Check last update time
4. Contact support if consistently wrong

**Problem:** Auto-mode not watering

**Check:**
1. Is auto-mode actually ON? (green toggle)
2. Is moisture above threshold?
3. Check next watering time
4. Verify irrigation policy settings

---

### Login Issues

**Problem:** Not receiving OTP

**Solutions:**
1. Check phone number is correct
2. Wait 2 minutes
3. Check SMS inbox and spam
4. Request OTP again
5. Try different phone number

**Problem:** OTP not accepted

**Solutions:**
1. Ensure 6-digit code
2. Check OTP hasn't expired (5 min validity)
3. Request new OTP
4. Clear browser cache

---

### Language Not Changing

**Problem:** Still shows wrong language

**Solutions:**
1. Click language toggle again
2. Refresh page
3. Clear browser cache
4. Check localStorage not blocked

---

### Weather Not Updating

**Problem:** Weather shows old data

**Solutions:**
1. Check internet connection
2. Refresh page
3. Check last update time (at bottom)
4. Verify location is correct
5. Weather updates every 30 minutes

---

### Market Prices Not Loading

**Problem:** Prices show "No data"

**Solutions:**
1. Check internet connection
2. Refresh page
3. Check if market is open (time of day)
4. Try different market/crop
5. Prices update daily at 9 AM

## FAQ

### General Questions

**Q: Is this app free?**
A: Yes, completely free for farmers.

**Q: Do I need to login?**
A: No, guest mode provides core features. Login for saving data and consultations.

**Q: Does this work offline?**
A: Partially. You can view cached data, but new scans/updates need internet.

**Q: What phones are supported?**
A: Any smartphone with a modern browser (Android 7+, iOS 12+).

**Q: Is my data private?**
A: Yes. Data is stored locally by default. Cloud sync only if you login.

---

### Disease Detection

**Q: How accurate is the AI?**
A: 70-90% accuracy depending on disease. Expert review improves accuracy.

**Q: Can I detect any disease?**
A: Currently trained on common crop diseases in Bangladesh. More being added.

**Q: What if AI is wrong?**
A: Request expert review. Doctor will provide correct diagnosis.

**Q: Can I scan multiple leaves?**
A: Yes, scan each separately for best results.

**Q: How many scans can I do?**
A: Unlimited scans for all users.

---

### Irrigation

**Q: Do I need special hardware?**
A: Yes, requires soil moisture sensor and irrigation controller (not included).

**Q: Can I control from anywhere?**
A: Yes, as long as you have internet connection.

**Q: What if I don't have auto irrigation?**
A: You can still monitor moisture and water manually.

**Q: How often is moisture updated?**
A: Every 15 minutes when auto-mode is ON.

---

### Consultations

**Q: Are consultations free?**
A: Demo version: Yes. Production: Doctor sets fee.

**Q: How long to get response?**
A: Usually within 24 hours. Urgent cases prioritized.

**Q: Can I choose my doctor?**
A: Yes, browse and choose from available experts.

**Q: What languages do doctors speak?**
A: All doctors speak Bengali. Many also speak English.

---

### For Doctors

**Q: How do I become a verified doctor?**
A: Register with valid credentials. Admin reviews and verifies (1-2 days).

**Q: What if verification is rejected?**
A: Admin provides reason. Fix issues and reapply.

**Q: Do I get paid for consultations?**
A: Demo: No. Production: Yes, through the platform.

**Q: Can I work flexible hours?**
A: Yes, set your availability schedule.
